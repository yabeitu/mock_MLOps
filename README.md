This repository contains the project for ITU BDS MLOPS'25

Collaborated on by Yasmine (yabe@itu.dk) and Luke (luon@itu.dk)

The given 'Inputs' are:
- python monolith, in the 'notebooks' folder
- raw input data, in the 'notebooks/artifacts' folder
- Github action to test model inference action, not within this repo, look at https://github.com/lasselundstenjensen/itu-sdse-project-model-validator

Required 'Outputs' are:
- This README.md file describing the project
- Github automation workflow
- Dagger workflow (in Go)
- Model artifact produced by Github workflow and named 'model'
